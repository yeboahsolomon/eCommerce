# E-Commerce Project

A full-stack e-commerce platform for online shopping.

## Features

- Product browsing and search
- Shopping cart
- User authentication
- Checkout and payment processing
- Order tracking
- Admin dashboard

## Installation

```bash
git clone https://github.com/yeboahsolomon/ecommerce-project.git
cd ecommerce-project
npm install
```

Create a `.env` file:
```
DATABASE_URL=your_database_url
JWT_SECRET=your_secret
```

Run the app:
```bash
npm start
```

## Tech Stack

- Frontend: NextJs
- Backend: Node.js/Express
- Database: Postgres
